Ancient Empires Reverse Engineered
Windows x64 release v0.1.0

This folder contains:

  AncientEmpires.exe        - player-facing source port
  AncientEmpiresEditor.exe  - research level editor

The executables include Python and their required libraries. No Python
installation is needed.

Original game assets are not included. Before launching either executable,
place your legally obtained copies of these files in the game_data folder:

  AEPROG.EXE
  AE000.DAT
  AE001.DAT

The game_data folder includes a README placeholder in public releases. You can
leave it there.

Version 0.1.0 provides the first room-local gameplay slice with recovered
rendering, HUD, player movement, collision and audio. The full original game is
still under active reconstruction; many systems and actor behaviors are not yet
complete.
